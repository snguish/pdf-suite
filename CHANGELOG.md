# Changelog

All notable changes to PDF Suite are documented here.

## [1.1.0] - 2026-07-20

### Added

- In-memory undo and redo history for document-changing operations.
- Selection, movement, resizing, and deletion for visual signatures placed in the current session.
- Toolbar history controls and an icon-based Tools rail.
- Illustrated in-app User Guide with keyboard reference.
- Responsive More Actions workspace for notes, forms, and signatures.
- Windows dark title bars for application popups.
- Complete workflow and signature rendering regression tests.

### Changed

- Refined menu, toolbar, spacing, responsive behavior, and version presentation.
- Expanded README documentation and feature gallery.
- Improved Windows release packaging with license and third-party notices.

### Fixed

- Prevented moved or resized signatures from rendering as empty boxes.
- Corrected persistent menu focus outlines and inconsistent application version labels.

### Security

- Clarified that visual signatures do not provide identity verification or document integrity.
